# Backend API

## Instalação

```bash
cd backend
npm install
```

## Rodar o servidor

```bash
npm start
```

O backend ficará disponível em http://localhost:3001

Certifique-se de que o MySQL está rodando e o banco de dados `advogados` está populado com as tabelas: `advogados`, `sociedades`, `estagiarios`, `tribunais`.
